## Key concepts learned

- **LocalDate / LocalDateTime**: Java에서 날짜와 시간을 다루는 클래스로, 현재 날짜/시간을 객체로 반환
- **StringTokenizer**: 지정된 구분자(delimiter)를 기준으로 문자열을 분할하여 토큰 단위로 처리
- **Wrapper Class**: 기본 자료형(int, double 등)을 객체로 감싸는 클래스로, 오토박싱/오토언박싱을 지원
- **enum**: 여러 상수를 관리하는 타입으로, 정의된 상수만 사용 가능하도록 제한
- **Exception Handling**: try-catch-finally를 사용하여 프로그램 비정상 종료를 방지하고 예외 상황 처리
- **ArrayIndexOutOfBoundsException**: 배열의 유효하지 않은 인덱스에 접근할 때 발생
- **NullPointerException**: null 참조에 대해 메서드를 호출할 때 발생
- **NumberFormatException**: 숫자가 아닌 문자열을 숫자로 변환하려 할 때 발생
- **ArithmeticException**: 0으로 나누기 같은 산술 연산 오류가 발생했을 때 발생

### Exception Handling - Basic Syntax

```javascript
try {
    // 예외가 발생할 수 있는 코드
} catch(예외타입 변수명) {
    // 예외 발생 시 실행
} finally {
    // 예외 발생 여부 상관없이 항상 실행
}
```

### Exception Handling - Example

```javascript
// ❌ 예외처리 없으면
int[] arr = {1, 2, 3};
System.out.println(arr[5]);  // 여기서 종료!
System.out.println("끝");    // 실행 안 됨!

// ✅ 예외처리 있으면
try {
    int[] arr = {1, 2, 3};
    System.out.println(arr[5]);  // 예외 발생!
} catch(ArrayIndexOutOfBoundsException e) {
    System.out.println("배열 범위를 벗어났습니다!");  // 사용자에게 알림
}
System.out.println("끝");  // 정상 실행! ✅
```

### Common Exception Types

```javascript
// NullPointerException
String s = null;
s.length();  // ❌ null인데 메서드 호출!

// NumberFormatException
Integer.parseInt("abc");  // ❌ 숫자가 아닌 문자열 변환!

// ArithmeticException
int n = 10 / 0;  // ❌ 0으로 나누기!
```

### finally Block

```javascript
try {
    System.out.println("try 실행");
    int n = 10 / 0;  // 예외 발생!
} catch(ArithmeticException e) {
    System.out.println("예외 발생!");
} finally {
    System.out.println("finally 항상 실행!");  // 무조건 실행!
}
// → try 실행
// → 예외 발생!
// → finally 항상 실행!
```
