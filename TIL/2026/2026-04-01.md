## Key concepts learned

- **Variable** : Storage Count 1, Primitive/Reference 타입 저장 가능
- **Array** : 고정 크기의 같은 타입 데이터만 저장 가능, resize 불가
- **Collection** : 자동 resize, 다양한 타입 저장 가능, Reference만 저장
- **List** : 순서가 있고 중복을 허용하는 Collection 타입
- **Set** : 순서가 없고 중복을 허용하지 않는 Collection 타입
- **Map** : Key/value 쌍으로 저장되고 빠른 검색이 가능한 Collection 타입
- **Wrapper Class** : 원시 타입을 참조 타입으로 감싸는 클래스 (ex. int -> Integer)
- **2D Array** : 배열 안에 배열을 포함하는 다차원 배열 구조
- **Arrays.toString()** : 배열을 출력 가능한 문자열로 변환하는 메서드
- **Integer.parseInt()** : String을 int로 변환하는 메서드
- **Float.parseFloat()** : String을 float로 변환하는 메서드
- **Math.random()** : 0.0부터 1.0 사이의 double 값 반환
- **Random class** : java.util.Random을 이용한 난수 생성

## Commands / Configuration

### Array Syntax

```java
// 1D Array
int[] num;                    // Declaration
num = new int[3];             // Creation
num[0] = 1;                   // Initialization
int[] num = {1, 2, 3};        // Literal (declaration과 동시에만 가능)
num = new int[] {1, 2, 3};    // Literal with new (선언 분리 가능)

// 2D Array
int[][] num;                  // Declaration
num = new int[2][];           // Row creation
num[0] = new int[2];          // Column creation
int[][] num = {{1,2},{3,4}}; // Literal
num = new int[][]{{1,2},{3,4}}; // Literal with new (size 생략)
int[][] num = {{1},{2,3},{4,5,6}}; // Jagged array
```

### Arrays Utility Class

```java
import java.util.Arrays;

int[] num = {5, 3, 1, 4, 2};
Arrays.sort(num);              // Sort
Arrays.toString(num);          // Convert to String
Arrays.copyOf(num, 3);         // Copy array
Arrays.fill(num, 0);           // Fill with value
```

### 2D Array Output

```java
// Enhanced for loop
for(int[] k : num) {
    for (int x : k) {
        System.out.println(x);
    }
}
```

### String to Number Conversion

```java
int num = Integer.parseInt("10");        // String -> int
float f = Float.parseFloat("3.14");      // String -> float
```

### Random Number Generation

```java
// Math.random()
double d = Math.random();                // 0.0 ~ 1.0
int num = (int)(Math.random() * 10) + 1; // 1 ~ 10

// Random class
import java.util.Random;
Random rand = new Random();
rand.nextInt(10);                        // 0 ~ 9
rand.nextInt(10) + 1;                    // 1 ~ 10
```
