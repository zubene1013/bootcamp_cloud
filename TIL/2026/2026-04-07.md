## Key concepts learned

- **Polymorphism(다형성)**: 부모 타입의 배열이나 변수에 자식 타입의 객체를 저장할 수 있는 기능
- **Package(패키지)**: 클래스 파일을 관리하기 위한 폴더와 유사한 구조로, 도메인을 거꾸로 작성하는 컨벤션 사용
- **Import**: 다른 패키지의 클래스를 사용하기 위해 필요하며, java.lang은 자동으로 import됨
- **Interface Multiple Implementation**: 하나의 클래스가 여러 개의 인터페이스를 동시에 구현 가능
- **Interface Multiple Inheritance**: 인터페이스 간에는 다중 상속이 가능
- **Method Parameter Polymorphism**: 부모 타입의 파라미터 하나로 모든 자식 타입의 객체를 받을 수 있음
- **Dynamic Typing**: 컴파일 시점의 타입과 실행 시점의 실제 타입이 다를 수 있는 현상
- **Casting(형변환)**: 부모 타입에서 자식 타입의 멤버에 접근하기 위해 필요한 명시적 타입 변환
- **instanceof 연산자**: 객체의 실제 생성된 타입을 확인하는 연산자로, 형변환 전에 안전성 검증에 사용
- **Abstract Class(추상 클래스)**: 미완성 클래스로 상속을 통해 완성하여 사용하며, abstract 메서드는 구현 없이 선언만 함
- **Abstract Method**: 구현 없이 선언만 된 메서드로, 상속받은 클래스에서 반드시 오버라이딩해야 함(강제성)
- **Interface Constants**: 인터페이스의 변수는 자동으로 public static final로 처리되는 상수
- **Interface Abstract Method**: 인터페이스의 일반 메서드는 자동으로 public abstract로 처리됨
- **Default Method**: 인터페이스에서 구현이 있는 메서드로 default 키워드 사용
- **Static Method in Interface**: 인터페이스의 static 메서드는 인터페이스명.메서드명()으로 호출

## Commands / Configuration

### Polymorphism - Array Usage

```javascript
Employee[] arr = new Employee[3];
arr[0] = new Employee("홍길동", 2000);
arr[1] = new Manager("유관순", 4000, "관리");
arr[2] = new Engineer("이순신", 4300, "자바");
```

### Object Array - All Data Types

```javascript
Object[] obj = new Object[3];
obj[0] = new Employee();
obj[1] = "문자열";
obj[2] = 123;
```

### Method Parameter Polymorphism

```javascript
public static void method(Employee e) { }

method(new Employee("홍길동", 2000));
method(new Engineer("이순신", 4300, "자바"));
method(new Manager("유관순", 4000, "관리"));
```

### Dynamic Typing

```javascript
Employee e = new Manager("유관순", 4000, "관리");
e.a();
```

### Casting

```javascript
Employee e = new Manager("유관순", 4000, "관리");
e.depart;  // ❌ Employee에 depart 없음

Manager m = (Manager) e;  // 자식타입으로 형변환
m.depart;                 // ✅ 접근 가능!
```

### instanceof 연산자

```javascript
Employee e = new Manager("유관순", 4000, "관리");

e instanceof Manager   // true  ✅
e instanceof Engineer  // false ❌

// 활용 - 형변환 전에 체크!
if (e instanceof Manager) {
    Manager m = (Manager) e;
}
```

### Interface Multiple Implementation

```javascript
public class A extends B implements interface1, interface2 {
    public void a() { }  // Must be overridden!
}
```

### Interface Multiple Inheritance

```javascript
public interface C extends interface1, interface2 { }
```
