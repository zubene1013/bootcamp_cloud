## Key concepts learned

- **Primitive Data Types**: Java의 8가지 기본 데이터 타입 (byte, short, int, long, float, double, char, boolean)으로, 메모리 크기와 범위가 고정됨
- **Number Systems in Java**: 진수 표기법 - 2진수(0b), 8진수(0), 16진수(0x)를 사용하여 숫자를 표현
- **Type Suffixes**: long은 L, float는 F를 붙여 컴파일러에게 타입을 명시해야 함
- **Escape Characters**: \n, \t, \", \', \\을 사용하여 특수 문자와 공백 처리
- **Reference Data Types**: Primitive이 아닌 String, Array 등 객체 기반의 데이터 타입
- **Git Merge Strategies**: Fast-Forward (선형 히스토리), 3-Way Merge (병합 커밋 생성), Rebase (히스토리 재정렬)
- **Git Tracking**: gitignore로 추적 제외 파일 지정 및 `git rm --cached`로 이미 추적된 파일 제거 가능
- **Reject Error Resolution**: 로컬과 원격 히스토리가 다를 때 원격을 fetch하여 병합 후 충돌 해결 필요

## Commands / Configuration

### Java Data Types

```java
// Integer Types
byte b = 127;           // 1byte (8bit)
short s = 32767;        // 2byte (16bit)
int i = 2147483647;     // 4byte (32bit)
long l = 9223372036854775807L;  // 8byte (64bit), L suffix required

// Number Systems
int binary = 0b1010;    // Binary notation
int octal = 012;        // Octal notation
int hex = 0xA;          // Hexadecimal notation

// Floating Point Types
float f = 3.14F;        // 4byte (32bit), F suffix required
double d = 3.14;        // 8byte (64bit)

// Character Type
char c = 'A';           // 2byte (16bit), single quotes required

// Boolean Type
boolean flag = true;    // 1byte (8bit)

// Escape Characters
System.out.println("Hello\nWorld");   // \n : New line
System.out.println("Tab\there");      // \t : Tab
System.out.println("Quote: \"Hello\""); // \" : Double quote
System.out.println("Single: \'A\'");  // \' : Single quote
System.out.println("Backslash: \\");  // \\ : Backslash
```

### Git Commands

```bash
# Basic Commands
git init                              # Create local repository
git add <file>                        # Add files to staging area
git commit -m "message"               # Create version
git commit --amend                    # Modify last commit
git log                               # Check version history
git status                            # Check repository status

# Log Options
git log --oneline                     # One commit per line
git log --oneline --graph --all       # Visualize branching
git log -p                            # Print changes
git log --relative-date               # Print elapsed time

# Diff Commands
git diff                              # Compare working directory and repository
git diff --staged                     # Compare staging area and repository
git diff HEAD                         # Compare both working and staging
git diff <hash1> <hash2>              # Compare between commits

# Reset and Restore
git reset --soft <commit>             # Cancel commit only
git reset --mixed <commit>            # Cancel commit and add
git reset --hard <commit>             # Cancel all changes
git restore <file>                    # Delete modifications
git restore --staged <file>           # Unstage file

# Branch Commands
git branch <name>                     # Create branch
git branch -m <name>                  # Rename branch
git branch -M <name>                  # Force rename branch
git branch -D <name>                  # Delete branch
git branch -r                         # View remote branches
git branch -a                         # View all branches
git merge-base main <branch>          # Check branching point

# Switch and Merge
git switch <branch>                   # Change branch
git switch -c <branch>                # Create and change branch
git merge <branch>                    # Merge branch
git merge -D <branch>                 # Delete after merge

# Remote Commands
git remote add origin <url>           # Connect remote repository
git remote                            # View remote aliases
git remote rm origin                  # Disconnect remote
git push origin --delete <branch>     # Delete remote branch
git push origin <branch>              # Push to remote
git pull                              # Fetch and merge
git fetch                             # Fetch without merge
git clone <url>                       # Clone repository
git clone <url> <folder>              # Clone with custom folder name
git clone -b <branch> <url>           # Clone specific branch
git clone --depth 1 <url>             # Shallow clone

# Commit and Add Combined
git commit -am "message"              # Add tracked files and commit

# Gitignore
git rm --cached <file>                # Remove already tracked file
```
