## Key concepts learned

- **Method Syntax**: public 반환타입 메서드명(매개변수) { 명령문; [반환값;] } 형태로 선언되며, 매개변수와 반환값의 유무에 따라 4가지 형식으로 분류됨
- **Static Variable**: 클래스 변수로 메서드 외부에서 선언되며, ClassName.variable 형태로 접근
- **Static Method**: 클래스에 속한 메서드로 ClassName.method() 형태로 호출하며, main 메서드도 static 메서드의 일종
- **Static Keyword in Nested Class**: static 키워드는 중첩 클래스(nested class)에서만 사용 가능
- **Call by Value & Call by Reference**: 메서드 호출 시 기본형은 값으로, 참조형은 주소로 전달되며 TestMain → Service → DAO → 데이터베이스 흐름에서 주고받음
- **메서드 4가지 형식**: 매개변수 없고 반환값 없음, 매개변수 있고 반환값 없음(setter), 매개변수 없고 반환값 있음(getter), 매개변수 있고 반환값 있음

## Commands / Configuration

### Method Syntax

```java
public 'return type' 'method name' ([parameter]) {
    sentence;
    [return value;]
}

// Usage examples
Test t = new Test();
t.method();                    // a. parameter X, return X
t.method(10, "홍길동");        // b. setter
int result = t.method();       // c. getter
int result = t.method(1,2);    // d. parameter O, return O
```

### Static Variable and Method

```java
public class A {
    static class B { }         // Only possible in nested class
    static int num;            // Static variable
    
    public static void method() {}
    public static void main(String[] args) {}
}

// How to use
ClassName.variable
ClassName.method()
Integer.parseInt("10");        // Common example
```
