## Key concepts learned

- **JOIN**: 원하는 데이터가 여러 테이블에 흩어져 있을 때 테이블을 연결해서 한 번에 조회하는 방법
- **NATURAL JOIN**: 같은 컬럼명을 자동으로 찾아서 테이블을 연결하는 방식
- **USING 절**: 연결할 공통 컬럼명을 직접 지정하여 테이블을 연결하는 방식
- **ON 절**: 조건을 직접 작성하여 테이블을 연결하는 방식 (가장 많이 사용)
- **다중 JOIN**: 3개 이상의 테이블을 한 번에 연결할 때 한 테이블이 허브 역할을 함
- **equi JOIN**: 같음(=) 조건으로 테이블을 연결하는 방식
- **non-equi JOIN**: BETWEEN 등 범위 조건으로 테이블을 연결하는 방식
- **LEFT OUTER JOIN**: 왼쪽 테이블의 모든 데이터를 기준으로 조인 (실무에서는 거의 LEFT JOIN만 사용)
- **RIGHT OUTER JOIN**: 오른쪽 테이블의 모든 데이터를 기준으로 조인
- **SELF JOIN**: 같은 테이블을 별칭만 다르게 해서 자기 자신과 연결하는 방식
- **서브쿼리**: SELECT 안에 SELECT를 넣어서 한 번의 쿼리로 원하는 데이터를 가져오는 방법 (반드시 괄호로 감싸야 함)
- **단일행 서브쿼리**: 서브쿼리 결과가 1개인 경우, 비교연산자(>, <, =, <=, >= 등) 사용
- **복수행 서브쿼리**: 서브쿼리 결과가 여러 개인 경우, IN / ANY / ALL 연산자 사용
- **IN 연산자**: 서브쿼리 결과 중 하나와 일치하는 행을 반환
- **ANY 연산자**: 서브쿼리 결과 중 하나라도 조건을 만족하면 해당 행을 반환
- **ALL 연산자**: 서브쿼리 결과의 모든 값이 조건을 만족해야 해당 행을 반환
- **EXISTS 연산자**: 서브쿼리 결과가 존재하면 메인쿼리를 실행, 없으면 실행하지 않음
- **인라인 뷰**: FROM 절에 서브쿼리를 넣어서 그 결과를 테이블처럼 사용하는 방식

## Commands / Configuration

### ANSI JOIN - NATURAL JOIN

```sql
SELECT *
FROM 테이블1 NATURAL JOIN 테이블2;
```

### ANSI JOIN - USING

```sql
SELECT *
FROM 테이블1 JOIN 테이블2 USING(공통컬럼명);
```

### ANSI JOIN - ON

```sql
SELECT ename, sal, dname, loc, e.deptno
FROM emp e JOIN dept d ON e.deptno = d.deptno;
```

### 3개 테이블 JOIN

```sql
SELECT *
FROM emp e JOIN dept d ON e.deptno = d.deptno
           JOIN salgrade s ON e.sal BETWEEN s.losal AND s.hisal;
```

### equi JOIN vs non-equi JOIN

```sql
-- equi JOIN
SELECT *
FROM emp e JOIN dept d ON e.deptno = d.deptno;

-- non-equi JOIN
SELECT *
FROM emp e JOIN salgrade s ON e.sal BETWEEN s.losal AND s.hisal;
```

### OUTER JOIN

```sql
-- 왼쪽 테이블 기준 (t1 데이터 모두 출력)
SELECT *
FROM t1 LEFT OUTER JOIN t2 ON 조건식;

-- 오른쪽 테이블 기준 (t2 데이터 모두 출력)
SELECT *
FROM t1 RIGHT OUTER JOIN t2 ON 조건식;
```

### SELF JOIN

```sql
-- 사원명과 관리자명 출력
SELECT e.ename AS 사원, m.ename AS 관리자
FROM emp e JOIN emp m ON e.mgr = m.empno;

-- 관리자가 없는 사원도 포함 (OUTER JOIN)
SELECT e.ename AS 사원, m.ename AS 관리자
FROM emp e LEFT JOIN emp m ON e.mgr = m.empno;
```
