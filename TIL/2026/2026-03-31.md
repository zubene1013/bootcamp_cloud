## Key concepts learned

- **Conditional Statement**: if, else if, else를 활용한 조건부 실행 구조
- **Block Scope**: 변수는 선언된 블록 내에서만 사용 가능하며, 블록 외부에서 사용하려면 외부에서 선언해야 함
- **Switch Statement**: switch(variable)를 사용한 다중 조건 분기, byte/short/int/char/String/enum 타입 지원, break 없으면 다음 case로 fall through
- **For Loop**: 반복 횟수가 예측 가능할 때 사용, 조건이 false면 실행 안 됨
- **While Loop**: 반복 횟수가 예측 불가능할 때 사용, 조건이 false면 실행 안 됨
- **Do-While Loop**: 최소 1회 이상 반드시 실행되는 루프, 조건 확인 후 실행
- **Break**: 루프를 완전히 탈출하는 제어문
- **Continue**: 현재 반복을 스킵하고 다음 반복으로 진행하는 제어문
- **Array Reference Type**: 배열은 참조 타입으로 주소값을 저장
- **Array Declaration and Creation**: 선언 시 주소값 저장, new 키워드로 힙 메모리에 실제 배열 생성

## Commands / Configuration

### Conditional Statements

```java
if (condition) {
    sentence
}

if (condition) {
    sentence1
} else {
    sentence2
}

if (condition1) {
    sentence
} else if (condition2) {
    sentence2
} else {
    sentence3
}
```

### Switch Statement

```java
switch(variable) {
    case value1:
        sentence;
        break;
    default:
        base sentence
}
```

### Loop Statements

```java
// For Loop
for (initialization; condition; increment) {
    sentence
}

// While Loop
while (condition) {
    sentence
}

// Do-While Loop
do {
    sentence
} while (condition);
```

### Break and Continue

```java
if (condition) break;

if (condition) continue;
```

### Array Declaration and Creation

```java
int[] n;  // Declaration - stores address value

n = new int[5];  // Creation - creates array in heap
```
