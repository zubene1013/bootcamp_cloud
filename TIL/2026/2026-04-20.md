## Key concepts learned

- **JDBC (Java DataBase Connectivity)**: 자바 프로그램과 DBMS를 연결해주는 기술로, DBMS 독립적이며 예외처리(SQLException, ClassNotFoundException)가 필수
- **JDBC 7단계 연동 흐름**: 접속 정보 설정 → 드라이버 로딩 → Connection 맺기 → SQL 작성 → PreparedStatement 얻기 → SQL 전송 → 자원 반납
- **PreparedStatement**: SQL을 DB로 전달하는 역할을 하며, executeQuery()는 SELECT, executeUpdate()는 DML(INSERT/UPDATE/DELETE) 실행
- **ResultSet**: SELECT 결과를 담는 객체로 rs.next()로 행을 순회하며 rs.getInt(), rs.getString() 등으로 데이터 추출
- **DTO (Data Transfer Object)**: DB에서 꺼낸 한 행의 데이터를 담는 그릇으로, 테이블 한 행 = DTO 객체 하나
- **DAO (Data Access Object)**: DB에 접근하는 코드만 모아둔 클래스로, Connection을 파라미터로 받으며 rs, pstmt만 close()하고 con은 닫지 않음
- **Service/ServiceImpl 패턴**: Service는 기능 정의 인터페이스, ServiceImpl은 드라이버 로딩과 Connection 생성을 담당하며 DAO 호출
- **setDao() 주입 패턴**: ServiceImpl의 dao 필드에 DAO 객체를 주입받아야 하며, null 상태에서 호출 시 NullPointerException 발생

```java
String driver = "com.mysql.cj.jdbc.Driver";
String url    = "jdbc:mysql://localhost:3306/testdb";
String userid = "*";
String passwd = "1234";
```

### 드라이버 로딩

```java
try {
    Class.forName(driver);
} catch (ClassNotFoundException e) {
    e.printStackTrace();
}
```

### Connection 맺기

```java
Connection con = DriverManager.getConnection(url, userid, passwd);
```

### SELECT 쿼리 실행

```java
String sql = "select deptno, dname, loc from dept";
PreparedStatement pstmt = con.prepareStatement(sql);
ResultSet rs = pstmt.executeQuery();
while (rs.next()) {
    int    deptno = rs.getInt("deptno");
    String dname  = rs.getString("dname");
    String loc    = rs.getString("loc");
}
```

### DML 쿼리 실행

```java
String sql = "insert into dept(deptno, dname, loc) values(50, '개발', '서울')";
PreparedStatement pstmt = con.prepareStatement(sql);
int n = pstmt.executeUpdate();
if (n >= 1) System.out.println("성공!");
```

### DTO 클래스 구성

```java
public class DeptDTO {
    int deptno;
    String dname;
    String loc;

    public DeptDTO() {}
    public DeptDTO(int deptno, String dname, String loc) {
        this.deptno = deptno;
        this.dname  = dname;
        this.loc    = loc;
    }

    public int getDeptno()             { return deptno; }
    public void setDeptno(int deptno)  { this.deptno = deptno; }
    public String getDname()           { return dname; }
    public void setDname(String dname) { this.dname = dname; }
    public String getLoc()             { return loc; }
    public void setLoc(String loc)     { this.loc = loc; }

    @Override
    public String toString() {
        return "DeptDTO [deptno=" + deptno + ", dname=" + dname + ", loc=" + loc + "]";
    }
}
```
