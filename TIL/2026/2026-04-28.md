## Key concepts learned

- **웹 아키텍처**: 클라이언트와 서버가 요청/응답을 주고받는 구조로, 서버는 클라이언트 요청을 처리하고 응답하는 역할을 함
- **정적 컴포넌트**: 웹서버에서 HTML 파일을 찾아 응답하는 방식으로 항상 동일한 화면을 보여줌
- **동적 컴포넌트**: 서버에서 요청할 때마다 실행하여 실행 결과를 HTML로 만들어 응답하므로 매번 결과가 달라짐
- **브라우저 캐시**: 한 번 다운로드한 파일은 캐시에 저장되어 같은 요청 시 캐시된 파일을 사용함
- **웹 표준**: 모든 브라우저에서 동일하게 동작하도록 W3C에서 정한 규칙
- **HyperText**: 링크로 연결되어 선택 시 새 화면으로 이동하는 텍스트
- **Markup Language**: 태그로 구성되어 특정 의미를 가지는 언어
- **DOM 트리**: 브라우저가 HTML을 읽어 계층적인 트리 구조로 만든 것으로, 태그, 속성, 텍스트가 모두 노드가 됨
- **DOCTYPE 선언**: 브라우저에게 HTML5 문법으로 처리하라고 알려주는 선언으로 파일 맨 위에 작성해야 함
- **전역 속성**: 모든 태그에 사용 가능한 속성(id, class, name, style)
- **지역 속성**: 특정 태그에서만 사용 가능한 속성(href, value 등)
- **커스텀 속성**: data-속성명="값" 형식으로 임의로 지정 가능한 속성
- **빈 태그**: 종료 태그가 없는 태그로 <br>, <img>, <input>, <hr> 등이 있음
- **CSS 설정 3가지**: 인라인(style 속성), 인터널(<style> 태그), 익스터널(<link> 태그)
- **JS 설정 2가지**: 내부(<script> 태그), 외부(<script src=""> 태그)
- **인라인 태그**: 가로 방향으로 배치되고 필요한 만큼의 공간만 차지하는 태그
- **블럭 레벨 태그**: 세로 방향으로 배치되고 전체 가로 공간을 차지하는 태그
- **기본 폰트 크기**: 16px (= 1rem)
- **a 태그의 target="_blank"**: 새 탭에서 링크를 열도록 지정
- **내부 링크**: 같은 페이지 내 특정 위치로 이동하기 위해 id 속성과 href="#id명" 사용
- **표 셀 병합**: colspan으로 열 병합(가로), rowspan으로 행 병합(세로)
- **순서 있는 목록 start 속성**: 원하는 번호부터 시작하도록 지정 가능

## Commands / Configuration

### HTML 기본 구조
```html
<!DOCTYPE html>
<html>
  <head>
    <title>페이지 제목</title>
  </head>
  <body>
    <!-- 화면에 표시될 내용 -->
  </body>
</html>
```

### CSS 설정 방법 3가지
```html
<!-- 1) 인라인 -->
<p style="color: red;">Hello</p>

<!-- 2) 인터널 -->
<style>
    p { color: red; }
</style>

<!-- 3) 익스터널 -->
<link rel="stylesheet" href="style.css">
```

### JS 설정 방법 2가지
```html
<!-- 내부(internal) -->
<script>
    alert("경고창");
</script>

<!-- 외부(external) -->
<script src="external.js"></script>
```

### 제목 태그
```html
<h1>가장 큰 제목 (32px)</h1>
<h4>중간 제목 (16px = 기본폰트)</h4>
<h6>가장 작은 제목</h6>
```
