## Key concepts learned

- **HTML 주석**: 브라우저에 표시되지 않는 메모용 코드, 단축키 Ctrl + /
- **img 태그**: src(경로), alt(대체 텍스트), width/height(크기) 속성, 빈 태그, 인라인 레벨
- **특수문자**: HTML에서 직접 입력 불가한 문자를 코드로 표현 (&nbsp;, &gt;, &lt;, &quot;, &copy;)
- **iframe 태그**: 현재 페이지 안에 다른 HTML 페이지를 삽입, 유튜브/지도 등 외부 콘텐츠 삽입 시 사용
- **dialog 태그**: HTML5 기본 모달 팝업, showModal()로 열고 method="dialog"로 자동 닫힘 처리
- **캐스케이딩(Cascading)**: DOM 트리 상위 요소 스타일이 하위로 전달되며 하위에서 재정의 시 덮어씀
- **명시도(Specificity)**: 선택자마다 우선순위 존재 — 인라인(1000) > ID(0100) > class(0010) > 태그(0001)
- **!important**: 모든 가중치를 무시하고 최우선 적용
- **계층 구조 선택자**: 자식(>), 자손( ), 인접 형제(+), 일반 형제(~) 구분
- **속성 선택자**: [attr], [attr="값"], [attr^="값"], [attr$="값"], [attr*="값"] 형식
- **의사 클래스(Pseudo-class)**: 특정 상태일 때 스타일 적용 (:hover, :focus, :nth-child 등)
- **의사 요소(Pseudo-element)**: 요소의 특정 부분에 스타일 적용 (::before, ::after, ::first-letter 등)
- **Box Model**: content → padding(안쪽 여백) → border(테두리) → margin(바깥 여백) 구조
- **box-sizing: border-box**: width/height에 border와 padding 포함, 실무에서 전체 적용 많이 함
- **margin 상쇄**: 블럭 요소의 위아래 margin이 겹치면 큰 값 하나만 적용됨
- **display 속성**: block / inline / inline-block / none으로 요소 성질 변경
- **inline-block**: 가로 배치 유지 + width/height 적용 가능 (두 가지 장점 결합)
- **position 속성**: static(기본) / relative(원래 위치 기준) / absolute(부모 기준) / fixed(브라우저 창 고정) / sticky
- **absolute 주의**: 부모에 position: relative 필수, 없으면 최상위(body)까지 올라가서 기준 잡음
- **z-index**: 요소 쌓임 순서 결정, position이 static이 아닐 때만 적용됨
- **Flexbox**: display: flex로 활성화, Container 속성(justify-content, align-items 등)과 Item 속성(flex-grow 등)으로 구성
- **Media Query**: 화면 크기에 따라 다른 스타일 적용, 반응형 웹의 핵심 기술

## Commands / Configuration

### HTML 주석

```html
<!-- 주석 내용 -->
```

### img 태그

```html
<img src="images/001.png" alt="이미지 설명">
<img src="images/001.png" width="100" height="100" alt="이미지">
```

### 특수문자

```html
hello&nbsp;&nbsp;&nbsp;world   <!-- 여러 칸 공백 -->
3 &gt; 4                       <!-- > -->
&quot;홍길동&quot;              <!-- " -->
```

### iframe / dialog

```html
<iframe src="target.html" frameborder="1" title="설명"></iframe>

<dialog id="myDialog">
    <form method="dialog">
        <button value="cancel">취소</button>
        <button value="default">확인</button>
    </form>
</dialog>
<button onclick="document.querySelector('#myDialog').showModal()">열기</button>
```

### CSS 명시도

```css
/* 우선순위: 인라인 > ID > class > 태그 */
#xxx { color: blue; }       /* 0100 */
.yyy { color: red; }        /* 0010 */
div  { color: blueviolet !important; } /* 강제 최우선 */
```

### CSS 선택자

```css
* { color: red; }           /* 전체 */
h1 { color: blue; }         /* 태그 */
#one { color: red; }        /* ID */
.select { color: red; }     /* class */

#one > h1 { color: red; }   /* 자식 */
#one h1 { color: red; }     /* 자손 */
h1 + p { color: red; }      /* 인접 형제 */
h1 ~ p { color: red; }      /* 일반 형제 */

input[type="text"] { border: 1px solid red; }  /* 속성 */

a:hover { color: red; }                         /* 의사 클래스 */
li:nth-child(2n) { background: yellow; }        /* 짝수 */

p::before { content: "★ "; }                   /* 의사 요소 */
::selection { background: yellow; }
```

### Box Model

```css
div {
    width: 200px;
    height: 100px;
}

* { box-sizing: border-box; }  /* 실무에서 전체 적용 */

margin: 10px 20px 30px 40px;  /* 상 우 하 좌 (시계방향) */
margin: 0 auto;                /* 수평 중앙 정렬 */

border: 1px solid red;
border-radius: 50%;            /* 원형 */
```

### display / position

```css
span { display: block; }         /* 인라인 → 블럭 */
a { display: inline-block; }     /* 가로 배치 + 크기 지정 */
div { display: none; }           /* 완전히 숨김 */

.box { position: relative; top: 10px; left: 20px; }
.fixed-nav { position: fixed; top: 0; width: 100%; }

.box1 { z-index: 1; }
.box2 { z-index: 10; }
```

### Flexbox

```css
.container {
    display: flex;
    flex-direction: row;
    justify-content: center;   /* 주축 정렬 */
    align-items: center;       /* 교차축 정렬 */
    flex-wrap: wrap;
}

.item1 { flex-grow: 1; }
.item2 { flex-grow: 4; }
```

### Media Query

```css
@media (max-width: 768px) {
    body { font-size: 14px; }
    .container { flex-direction: column; }
}
```
