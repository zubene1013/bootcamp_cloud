## Key concepts learned

- **collection 태그** - 1:N 관계에서 부모 DTO 안에 자식 DTO List를 포함시키며, ofType 속성으로 List 요소 타입을 지정
- **association vs collection** - association은 1:1 관계에서 단일 객체를 javaType으로 매핑하고, collection은 1:N 관계에서 List를 ofType으로 매핑
- **DAO/Service 패턴** - DAO는 SqlSession으로 SQL 실행만 담당하고, Service는 SqlSession 열기/닫기와 commit 관리를 담당
- **SqlSession 관리** - SELECT는 commit 불필요하지만 INSERT/UPDATE/DELETE는 commit 필수이며, session.close()는 반드시 finally 블록에서 실행
- **MySqlSessionFactory** - configuration.xml을 읽어서 SqlSession을 생성하는 공장 클래스이며, sqlSessionFactory를 static 블록 밖에 선언해야 getSession()에서 접근 가능
- **configuration.xml vs Mapper.xml** - 두 파일 모두 MyBatis XML 설정 파일이지만 DOCTYPE이 다르며, configuration.xml은 DB 연결 설정, Mapper.xml은 SQL 쿼리 저장
- **변수의 스코프** - 블록 안에서 선언된 변수는 블록 밖에서 접근 불가능하므로 static 블록 밖에서 변수 선언 필요
- **DTO 자동 매핑** - 테이블 컬럼명과 DTO 변수명을 일치시키면 MyBatis가 자동으로 매핑

## Commands / Configuration

### MyBatis Collection 매핑

```xml
<resultMap id="empdeptjoinMap2" type="DeptDTO">
  <id property="deptno" column="deptno"/>
  <result property="dname" column="dname"/>
  <result property="loc" column="loc"/>
  <collection property="empList" ofType="EmpDTO">
    <id property="empno" column="empno"/>
    <result property="ename" column="ename"/>
    <result property="sal" column="sal"/>
  </collection>
</resultMap>

<select id="empdeptjoin2" resultMap="empdeptjoinMap2">
  SELECT * FROM emp JOIN dept ON dept.deptno = emp.deptno
  ORDER BY empno DESC
</select>
```

### DAO/Service 패턴 구현

```java
@Override
public List<DeptDTO> findAll() {
    SqlSession session = MySqlSessionFactory.getSession();
    List<DeptDTO> list = null;
    try {
        list = dao.findAll(session);
    } finally {
        session.close();
    }
    return list;
}

@Override
public int insert(DeptDTO dto) {
    SqlSession session = MySqlSessionFactory.getSession();
    int n = 0;
    try {
        n = dao.insert(session, dto);
        session.commit();
    } finally {
        session.close();
    }
    return n;
}
```

### MySqlSessionFactory 구현

```java
package com.config;

import java.io.IOException;
import java.io.InputStream;
import org.apache.ibatis.io.Resources;
import org.apache.ibatis.session.SqlSession;
import org.apache.ibatis.session.SqlSessionFactory;
import org.apache.ibatis.session.SqlSessionFactoryBuilder;

public class MySqlSessionFactory {

    static SqlSessionFactory sqlSessionFactory;

    static {
        String resource = "com/config/configuration.xml";
        InputStream inputStream = null;
        try {
            inputStream = Resources.getResourceAsStream(resource);
        } catch (IOException e) {
            e.printStackTrace();
        }
        sqlSessionFactory = new SqlSessionFactoryBuilder().build(inputStream);
    }

    public static SqlSession getSession() {
        SqlSession session = sqlSessionFactory.openSession();
        return session;
    }
}
```
