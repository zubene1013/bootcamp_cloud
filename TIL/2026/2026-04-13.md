## Key concepts learned

- **Consumer** - 파라미터를 받아서 소비만 하고 반환값이 없는 함수형 인터페이스
- **Supplier** - 파라미터 없이 값을 공급(반환)하는 함수형 인터페이스
- **Function** - 파라미터를 받아서 변환 후 반환하는 함수형 인터페이스
- **Operator** - 같은 타입의 입력을 받아 같은 타입으로 반환하는 함수형 인터페이스
- **Predicate** - 파라미터를 받아서 boolean 값을 반환하는 함수형 인터페이스
- **andThen** - 함수형 인터페이스를 연쇄 실행하는 메서드 (앞의 결과를 뒤로 전달)
- **Stream API** - 컬렉션 데이터를 필터링, 정렬, 통계 처리하는 방식
- **Collectors** - Stream의 최종처리에서 데이터를 List, Set, 통계값으로 변환하는 클래스
- **Optional** - null 값을 안전하게 다루고 기본값을 제공하는 클래스

## Commands / Configuration

### Consumer

```javascript
Consumer<String> c = s -> System.out.println(s);
c.accept("Hello");

BiConsumer<String, Integer> c2 = (s, n) -> System.out.println(s + "\t" + n);
c2.accept("홍길동", 10);
```

### Supplier

```javascript
Supplier<String> s = () -> "홍길동";
s.get();  // → 홍길동

DoubleSupplier s2 = () -> 3.14D;
s2.getAsDouble();  // → 3.14
```

### Function

```javascript
Function<String, Integer> f = t -> t.length();
f.apply("hello");  // → 5
```

### Operator

```javascript
BinaryOperator<String> x = (t, u) -> t + u;
x.apply("hello", "world");  // → helloworld

IntUnaryOperator x2 = y -> y + 100;
x2.applyAsInt(100);  // → 200
```

### Predicate

```javascript
Predicate<String> p = t -> t.length() == 3;
p.test("홍길동");  // → true
```

### Predicate 논리 메서드

```javascript
IntPredicate x = n -> n % 2 == 0;   // 2의 배수
IntPredicate x2 = n -> n % 3 == 0;  // 3의 배수

x.and(x2).test(6);    // AND → true
x.or(x2).test(16);    // OR  → true
x.negate().test(16);  // NOT → false
```

### andThen

```javascript
// Consumer andThen → 같은 값 전달
c.andThen(c2).accept("hello");
// → first 작업: hello
// → Second 작업: hello

// Function andThen → 앞의 리턴값을 뒤의 파라미터로 전달
// ⚠️ f1 리턴타입 = f2 파라미터타입 반드시 일치!
Function<String, Integer> f1 = s -> s.length();
Function<Integer, Integer> f2 = n -> n + 10;
f1.andThen(f2).apply("hello");  // → 15
```

### Stream API 생성

```javascript
// List에서
List<String> list = new ArrayList<>();
Stream<String> stream = list.stream();

// 배열에서
String[] names = {"이순신1", "이순신2"};
Stream<String> stream2 = Arrays.stream(names);
```

### Stream API 기본 처리

```javascript
// forEach (최종처리) → Consumer 사용
stream.forEach(System.out::println);

// distinct (중간처리) → 중복 제거
list.stream()
    .distinct()
    .forEach(System.out::println);
```

### 정렬

```javascript
// 오름차순
list.stream().sorted().forEach(System.out::println);

// 내림차순
list.stream().sorted(Comparator.reverseOrder()).forEach(System.out::println);

// 객체 특정 값으로 정렬
list.stream()
    .sorted(Comparator.comparing(Student::getGrade))         // 오름차순
    .sorted(Comparator.comparing(Student::getGrade, Comparator.reverseOrder())) // 내림차순
    .forEach(System.out::println);
```

### Skip / Limit

```javascript
// skip → 앞에서 n개 건너뜀
list.stream().skip(2).forEach(System.out::println);

// limit → 앞에서 n개만 가져옴
list.stream().limit(4).forEach
