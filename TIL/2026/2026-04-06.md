## Key concepts learned

- **Class Relationship (has-a)**: 한 클래스가 다른 클래스의 객체를 멤버 변수로 가지는 관계
- **Class Relationship (is-a)**: 상속을 통한 클래스 간의 계층 관계
- **Overloading**: 동일한 메서드 이름으로 매개변수의 개수, 타입, 순서를 다르게 하여 여러 메서드 정의
- **Overriding**: 자식 클래스가 부모 클래스의 메서드를 같은 이름, 매개변수, 반환타입으로 재정의
- **Object 클래스**: 모든 클래스의 최상위 부모 클래스로, 기본 toString() 메서드는 주소값 출력
- **toString() 메서드**: 참조변수 출력 시 자동으로 호출되는 Object 클래스의 메서드
- **super() 생성자**: 자식 클래스 생성자에서 부모 클래스 생성자를 호출하며, 반드시 첫 번째 라인에 위치해야 함
- **Polymorphism (다형성)**: 부모 타입의 변수로 모든 자식 객체를 관리할 수 있는 특성

## Commands / Configuration

### Overloading 예시

```javascript
public void print() { }
public void print(int n) { }
public void print(String s) { }
public void print(int n, String s) { }
```

### Overriding 예시

```javascript
public class Animal {
    public void cry() {
        System.out.println("...");
    }
}

public class Cat extends Animal {
    @Override
    public void cry() {
        System.out.println("야옹");
    }
}
```

### super() 생성자 호출

```javascript
public class Animal {
    String name;
    public Animal(String name) {
        this.name = name;
    }
}

public class Cat extends Animal {
    int age;
    public Cat(String name, int age) {
        super(name);
        this.age = age;
    }
}
```

### toString() 메서드

```javascript
public String toString() {
    return 패키지명.클래스명@16진수;
}

Cat c = new Cat();
System.out.println(c);
System.out.println(c.toString());
```

### Polymorphism 예시

```javascript
A a = new A();
a = new B();
a = new C();

B b = new A();
```
