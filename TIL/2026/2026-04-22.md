## Key concepts learned

- **라이브러리 vs 프레임워크** - 라이브러리는 외부 클래스 파일들의 묶음(*.jar)이고, 프레임워크는 jar 파일과 개발 틀을 함께 제공
- **MyBatis** - JDBC 사용을 편리하고 효율적으로 기능이 추가된 SQL Mapping Framework로, 개발자가 메서드를 호출하면 해당하는 SQL문을 자동으로 실행
- **jdbc.properties** - DB 연결정보(driver, url, userid, passwd)를 Java 코드 밖으로 분리해서 관리하는 파일
- **configuration.xml** - MyBatis 전체 설정 파일로 DB 연결 정보와 Mapper 파일들을 등록
- **DeptDTO** - 테이블의 레코드 1개를 담는 클래스로, 기본 생성자와 getter/setter가 필수
- **DeptMapper.xml** - SQL 쿼리를 보관하는 파일로 resultType, parameterType, namespace, id 속성을 통해 SQL을 정의
- **MySqlSessionFactory** - configuration.xml을 읽어서 SqlSession을 생성하는 공장 클래스로, static 블록에서 클래스 최초 로딩 시 한 번만 실행
- **SqlSession** - 실제 SQL을 실행하는 객체로 selectOne(), selectList() 메서드 제공
- **parameterType** - int, DTO 객체, HashMap 등으로 SQL에 파라미터를 전달하는 방식
- **#{변수명}** - parameterType에서 꺼내온 값으로 자동 바인딩되는 MyBatis 문법

## Commands / Configuration

### jdbc.properties 작성

```
# 주석문
# key=value 형식으로 작성
jdbc.driver=com.mysql.cj.jdbc.Driver
jdbc.url=jdbc:mysql://localhost:3306/testdb
jdbc.userid=root
jdbc.passwd=1234
```

### configuration.xml 설정

```xml
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE configuration PUBLIC "-//mybatis.org//DTD Config 3.0//EN"
  "http://mybatis.org/dtd/mybatis-3-config.dtd">

<configuration>

  <!-- jdbc.properties 파일 불러오기 -->
  <properties resource="com/config/jdbc.properties"/>

  <environments default="development">
    <environment id="development">
      <transactionManager type="JDBC"/>
      <dataSource type="POOLED">
        <property name="driver"   value="${jdbc.driver}"/>
        <property name="url"      value="${jdbc.url}"/>
        <property name="username" value="${jdbc.userid}"/>
        <property name="password" value="${jdbc.passwd}"/>
      </dataSource>
    </environment>
  </environments>

  <!-- Mapper 파일 등록 -->
  <mappers>
    <mapper resource="com/config/DeptMapper.xml"/>
  </mappers>

</configuration>
```

### DeptDTO.java

```java
package com.dto;

public class DeptDTO {

    int deptno;
    String dname;
    String loc;

    // 기본 생성자 (필수)
    public DeptDTO() {}

    // 매개변수 생성자
    public DeptDTO(int deptno, String dname, String loc) {
        this.deptno = deptno;
        this.dname = dname;
        this.loc = loc;
    }

    // toString()
    @Override
    public String toString() {
        return "DeptDTO [deptno=" + deptno + ", dname=" + dname + ", loc=" + loc + "]";
    }

    // getter/setter (필수)
    public int getDeptno() { return deptno; }
    public void setDeptno(int deptno) { this.deptno = deptno; }
    public String getDname() { return dname; }
    public void setDname(String dname) { this.dname = dname; }
    public String getLoc() { return loc; }
    public void setLoc(String loc) { this.loc = loc; }
}
```
