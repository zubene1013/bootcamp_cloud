## Key concepts learned

- **DML (Data Manipulation Language)**: 데이터 조작어로 레코드 추가, 수정, 삭제 작업을 수행하는 SQL 명령어
- **트랜잭션 (Transaction)**: 1개 이상의 DML 작업들을 논리적으로 하나의 작업단위로 묶어 처리하는 개념
- **DDL (Data Definition Language)**: 데이터 정의어로 테이블 생성, 수정, 삭제 작업을 수행하는 SQL 명령어
- **ON DELETE CASCADE**: FOREIGN KEY 옵션으로 부모 테이블의 데이터 삭제 시 자식 테이블의 해당 데이터도 자동 삭제
- **ON DELETE SET NULL**: FOREIGN KEY 옵션으로 부모 테이블의 데이터 삭제 시 자식 테이블의 FK 컬럼을 NULL로 변경
- **컬럼레벨 제약조건**: CREATE TABLE 시 각 컬럼을 정의하면서 바로 제약조건을 지정하는 방식
- **테이블레벨 제약조건**: 모든 컬럼을 정의한 후 마지막에 제약조건을 추가하는 방식
- **CTAS (Create Table As Select)**: SELECT 쿼리의 결과로 새 테이블을 생성하는 방법
- **TRUNCATE vs DELETE**: TRUNCATE는 데이터만 빠르게 삭제하고 구조 유지, DELETE는 행 단위로 삭제 가능

## Commands / Configuration

### Autocommit 설정

```sql
-- autocommit 확인
SHOW VARIABLES LIKE 'autocommit%';

-- autocommit 비활성화 (트랜잭션 직접 관리)
SET autocommit = false;
```

### INSERT 문

```sql
-- 컬럼명 지정 방식 (권장)
INSERT INTO 테이블명 (컬럼명, 컬럼명2)
VALUES (값, 값2);

-- 컬럼명 생략 방식
INSERT INTO 테이블명 VALUES (값, 값2, ..., null);

-- 서브쿼리로 다중 행 입력
INSERT INTO 테이블명 (컬럼명)
SELECT 컬럼명 FROM 테이블명;

-- VALUE LIST로 다중 행 입력
INSERT INTO 테이블명 (컬럼명)
VALUES (값1), (값2), (값3);

-- CTAS — 테이블 구조만 복사 (데이터 없이)
CREATE TABLE 새테이블명 AS
SELECT 컬럼명 FROM 테이블명
WHERE 1=2;
```

### UPDATE 문

```sql
-- 조건 없는 UPDATE (전체 수정)
UPDATE 테이블명
SET 컬럼명=새값;

-- 조건 있는 UPDATE (일반적인 방법)
UPDATE 테이블명
SET 컬럼명=새값
WHERE 조건식;
```

### DELETE 문

```sql
-- 조건 없는 DELETE (전체 삭제)
DELETE FROM 테이블명;

-- 조건 있는 DELETE (일반적인 방법)
DELETE FROM 테이블명
WHERE 조건식;
```

### CREATE TABLE

```sql
CREATE TABLE IF NOT EXISTS 테이블명
(
    컬럼명 데이터타입 [제약조건],
    컬럼명 데이터타입 AUTO_INCREMENT,
    컬럼명 데이터타입 DEFAULT 값,
    컬럼명 데이터타입 NOT NULL,
    컬럼명 데이터타입 REFERENCES master테이블(master컬럼) ON DELETE CASCADE,
    컬럼명 char(4) CONSTRAINT CHECK (컬럼명 IN ('M','F'))
);

-- 게시판 테이블 예시
CREATE TABLE IF NOT EXISTS board
(
    num      int          PRIMARY KEY AUTO_INCREMENT,
    title    varchar(100) NOT NULL,
    author   varchar(10)  NOT NULL,
    content  varchar(500) NOT NULL,
    writeday datetime     DEFAULT now(),
    gender   char(4)      CONSTRAINT CHECK (gender IN ('M','F')),
    readcnt  int          DEFAULT 0
);
```

### 컬럼레벨 제약조건 설정

```sql
CREATE TABLE IF NOT EXISTS board
(
    num    int          PRIMARY KEY AUTO_INCREMENT,
    title  varchar(100) NOT NULL,
    author varchar(10)  UNIQUE
);
```
