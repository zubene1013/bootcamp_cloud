## Key concepts learned

- **namespace 전체 경로 호출**: Mapper가 여러 개일 때 id 중복을 방지하기 위해 `namespace.id` 형식으로 호출하는 방식
- **typeAlias**: DTO 클래스의 전체 경로를 짧은 이름으로 등록하여 Mapper XML에서 간편하게 사용할 수 있는 기능
- **패키지 단위 typeAlias 등록**: `<package>` 태그로 특정 패키지의 모든 클래스를 자동으로 별칭 등록
- **@Alias 어노테이션**: DTO 클래스에 직접 별칭을 지정하는 방법
- **CDATA 블록**: XML에서 <, > 등 특수문자를 순수 문자열로 처리하기 위한 블록
- **DML 작성 및 commit()**: INSERT, UPDATE, DELETE 실행 후 반드시 commit()을 호출해야 DB에 실제로 반영됨
- **foreach 태그**: List를 반복하여 동적으로 SQL을 생성하는 태그 (멀티 INSERT, DELETE, SELECT에 사용)
- **동적 SQL (<if>)**: 조건에 따라 WHERE 절이 추가되거나 생략되는 SQL 작성
- **동적 SQL (<set>)**: UPDATE 문에서 값이 있는 컬럼만 동적으로 업데이트하는 방식
- **동적 SQL (<choose>)**: Java의 switch문처럼 여러 조건 중 하나를 선택하여 실행하는 방식
- **단일 컬럼 조회**: DTO가 아닌 int, String 등 기본 타입 하나를 반환할 때 resultType에 기본 타입 지정

## Commands / Configuration

### typeAlias 등록 - configuration.xml

```xml
<!-- 방법 1: 개별 등록 -->
<typeAliases>
  <typeAlias alias="DeptDTO" type="com.dto.DeptDTO"/>
</typeAliases>

<!-- 방법 2: 패키지 전체 등록 (권장) -->
<typeAliases>
  <package name="com.dto"/>
</typeAliases>
```

### @Alias 어노테이션

```java
import org.apache.ibatis.type.Alias;

@Alias("EmpDTO")
public class EmpDTO {
    ...
}
```

### 여러 Mapper 등록 - configuration.xml

```xml
<mappers>
  <mapper resource="com/config/DeptMapper.xml"/>
  <mapper resource="com/config/DeptMapper2.xml"/>
  <mapper resource="com/config/EmpMapper.xml"/>
</mappers>
```

### namespace 전체 경로 호출

```java
// Day 1 방식 (Mapper 하나일 때)
session.selectOne("findByDeptno", 20);

// Day 2 방식 (Mapper 여러 개일 때 권장)
session.selectOne("com.config.DeptMapper.findByDeptno", 20);
```

### CDATA 블록 사용

```xml
<!-- ❌ 오류 -->
<select id="findByDeptnoLessThan" resultType="DeptDTO" parameterType="int">
  SELECT deptno, dname, loc FROM dept
  WHERE deptno < #{deptno}
</select>

<!-- ✅ 해결 -->
<select id="findByDeptnoLessThan" resultType="DeptDTO" parameterType="int">
  <![CDATA[
    SELECT deptno, dname, loc FROM dept
    WHERE deptno < #{deptno}
  ]]>
</select>
```

### INSERT

```xml
<insert id="insert" parameterType="DeptDTO">
  INSERT INTO dept (deptno, dname, loc)
  VALUES (#{deptno}, #{dname}, #{loc})
</insert>
```

```java
DeptDTO data3 = new DeptDTO(61, "개발", "서울");
int n = session.insert("com.config.DeptMapper.insert", data3);
if(n >= 1) System.out.println("저장 성공");
session.commit();
```
