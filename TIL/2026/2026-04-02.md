## Key concepts learned

- **Object** : Attribute(특성)와 Behavior(행동)로 구성되는 실제 세계의 개체
- **Class** : 객체를 가상 세계에서 표현하는 방식으로 Instance Variable과 Method로 구성
- **Abstraction** : 필요한 객체, 속성, 행동을 추출하는 프로세스
- **Object Creation Flow** : Stack에 Reference variable 생성 → Heap에 Instance 생성 → Constructor 호출 → 주소값 저장
- **Access Modifier** : public(모든 곳), protected(같은 패키지+상속), default(같은 패키지), private(같은 클래스) 네 가지 접근 범위 제어
- **Constructor** : new 키워드로 호출되며 instance variable을 초기화하는 메서드
- **Constructor Overloading** : 인자 개수, 타입, 순서가 다르면 여러 constructor 정의 가능
- **Default Constructor** : custom constructor가 없을 때만 자동 제공됨
- **this Keyword** : 같은 이름의 변수를 구분하거나 다른 constructor를 호출할 때 사용
- **Method** : Instance variable을 저장(private 권장)하고 다양한 기능을 처리하는 역할

## Commands / Configuration

### Constructor Examples

```java
class Cat {
    // Instance Variable (Attribute)
    String name;
    int age;
    
    // Default Constructor
    public Cat() {}
    
    // Parameter Constructor
    public Cat(String name, int age) {
        this.name = name;
        this.age = age;
    }
    
    // Method (Behavior)
    void eat() {}
}
```

### Access Modifier

```java
public class Example {
    public int publicVar;           // Accessible from anywhere
    protected int protectedVar;     // Same package + inheritance
    int defaultVar;                 // Same package only
    private int privateVar;         // Same class only
}
```
